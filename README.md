# Tableau
Global Mental Health Impact Dashboard

📘 Overview

This project presents an interactive Tableau dashboard that analyzes global mental health trends using data imported from Excel. The dashboard explores how mental health conditions vary across countries, industries, genders, and severity levels, offering a clear and intuitive understanding of global mental health patterns. The dashboard is designed to support awareness, comparison, and exploration of mental health impacts across different demographics and regions.

📊 Features

* Interactive global map showing country‑wise mental health impact
* Industry‑wise bar chart comparing affected populations across job sectors
* Gender distribution donut chart
* Severity‑based area chart comparing high, medium, and low cases across countries
* Dynamic filters for age range and occupation
* Fully interactive dashboard where selecting any element updates all visuals

📁 Dataset

The dashboard uses a structured Excel dataset containing:
* Country
* Gender
* Occupation
* Severity level (Low, Medium, High)
* Count of affected individuals
The dataset is anonymized and includes diverse demographic and occupational groups.

📈 Visuals Included

1. Global Distribution Map
A world map showing the number of mentally affected individuals per country.

2. Industry‑Wise Bar Chart
Compares mental health impact across sectors such as Healthcare, Engineering, IT, Finance, Sales, and Education.

3. Gender Distribution Donut Chart
Shows the percentage distribution across Female, Male, Non‑binary, and Prefer‑not‑to‑say categories.

4. Severity‑Based Area Chart
Displays high, medium, and low severity cases across seven countries.

🔗 Interactivity

The dashboard is fully interactive:

* Clicking a country filters all visuals to that country
* Selecting an industry updates gender and severity charts
* Age range and occupation filters refine all insights


🧾 Conclusion

This Tableau dashboard provides a comprehensive and engaging view of global mental health trends. By combining geography, industry, gender, and severity insights with interactive filters, it enables deeper exploration and supports awareness and targeted intervention strategies.
